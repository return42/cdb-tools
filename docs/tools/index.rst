================================================================================
Werkzeuge der CDB-Tools
================================================================================

Die hier vorgestellten Werkzeuge stehen in einer CDB-Tools Umgebung zur
Verfügung. Eine solche Umgebung (Shell) bekommt man z.B. durch Doppelklick auf
:ref:`tools_sh_bat`. In der CDB-Tools Umgebung stehen alle Kommandos aus der CDB
Software als auch aus den CDB-Tools zur Verfügung. Also auch alles was mittels
:ref:`pip --user <cdbtools_pckg>` installiert wurde (s.a. `requirements.txt`_).
