=========
WEB-Links
=========

Die CDB-Links verweisen auf WEB-Seiten eines CDB Servers, wie er z.B. mittels
``cdb-localhost-START`` gestartet werden kann (alles *localhost*).

- ``cdb-localhost-Portal.url`` : http://localhost:81/
- ``cdb-localhost-Doc.url``    : http://localhost:81/doc
- ``cdb-localhost-Services``   : http://localhost:55550/services
